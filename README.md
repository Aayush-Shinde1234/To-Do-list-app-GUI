# 📝 Tkinter To-Do List App

A simple desktop To-Do List application built using **Python Tkinter**.  
It allows users to add, remove, view, and delete tasks using a graphical interface.

---

## 🚀 Features

- ➕ Add new tasks
- ❌ Remove selected task
- 🔄 Refresh task list
- 🧹 Delete all tasks
- 💾 Tasks are saved in a local `tasks.txt` file (persistent storage)

---

## 🖥️ Preview

> A simple GUI window with:
- Entry box to type tasks
- Buttons for actions (Add, Remove, Refresh, Delete All)
- Listbox showing all saved tasks
